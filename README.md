# Moliverse for French study in China

莫莉宇宙——面向中国法语学习者的文字游戏项目。本质上是一款面向中国孩子的小语种启蒙产品，目前落地的是法语这一门。但它不是传统的背单词 App，而是一个剧情驱动、靠选择推进的互动叙事游戏——形式上接近视觉小说 + 文字冒险，机制上明显借鉴了《极乐迪斯科》：选项分三岔（保守推进 / 技能检定 / 文化碰撞），属性被拟人化成"内心小人"在脑中说话，配合词卡排序、听力、闪卡几种小游戏来消化词汇。
它最聪明的设定在视角：玩家不是那个"来学语言的外国人"，而是中国东道主小女孩雯雯。法国女孩 Emilie 通过"中法小使者"项目来中国，住进雯雯家，而一只会心灵感应、能"发光翻译"语言与情绪的阿尔卑斯魔法小鹿 Moli 全程陪伴。于是学法语这件事被包装成了"我要用法语帮我的外国朋友、当好这个主人"——是双向的跨文化友谊，而不是单向灌输。故事横跨三站：上海 → 广东 → 云南，共九个单元，一路遇见弄堂早餐、早茶叩手礼、舞狮咏春、大理扎染、西双版纳大象、哈尼梯田，最后机场告别。
真正让它区别于普通语言 App 的，是底下那两层。一层是德育与文化：每个单元都挂着一个核心品德（勇毅、和谐、尊重食物、武德、仁爱、友谊……），配古文出处（论语、庄子、孟子、费孝通、王勃）和一张"思想卡"故事，再加一段**"不同但有趣、和而不同"的中法对比**（牛角包配咖啡 vs 油条配豆浆、凡尔赛对称花园 vs 豫园曲径）——明显是冲着"立德树人 / 中华优秀传统文化进校园"去的。另一层是成长系统：开局用儿童版 MBTI 自动匹配三种人格（破局者 / 解谜家 / 倾听者），配六维成长轨道、Moli 币、正向无惩罚、零付费无攀比、家长可控时长——整套都是按 3–8 岁少儿产品的合规要求设计的。
从工程上看，它还设计了一条半 AI 生成的内容流水线：一个"主笔 AI"按格式生成剧本 JSON，一个"档案与规则智能体（EF Agent）"做冷冰冰的数值/逻辑仲裁（检定阈值不能超玩家等级 +3、不许凭空捏造道具、压力值结算要合规），再加一本《审核必读》手册从致命错误、视觉光影、玩法逻辑、叙事质感、教研评估五道防线把关。也就是说，它不只是一个游戏，更像一个可规模化产出剧情的教育内容系统。
